Pilihan Soal

Instruksi Soal

  1. Buat dulu branch anda dari branch master
  2. Ketentuan nama branch adalah: quiz/login_NIM
  3. Lokasi project kalian beradan di folder quiz/02_login/NIM
  4. Buat database sesuai dengan rancangan tabel yang sudah di tentukan
  5. Buat login dengan 2 role dan 2 akun login, "admin", "user"
  6. Jika login sbg admin, maka redirect ke halaman admin jika user ke halaman user
  7. Lakukan validasi halaman
  8. Lakukan PR dengan nama branch tadi dan jangan lupa untuk melampirkan hasil output dari program
  9. Jangan lupa di PR nya untuk menambahkan saya sebagai reviewer, ada di bagian kanan ketika PR
